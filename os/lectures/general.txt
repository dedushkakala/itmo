Системные библиотеки и user space хранится в ABI(application binary interface).

3)VM(Virtual Memory - организация виртуальной памяти)
4)NET(работа с сетью)
5)DRIVERS(Драйвера для железа)
6)VIRT(virtualization)
7)ARCH(специфичный для архитектуры код)
